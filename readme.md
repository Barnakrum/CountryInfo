# Country Info

Website built using React, TypeScript, Tailwindcss, React.Query. It displays most important info about every country. It uses [restcountries.com](http://restcountries.com "restcountries.com"), [United Nations](http://population.un.org/dataportalapi/index.html "United Nations"), [openweathermap.org](https://openweathermap.org/ "openweathermap.org") APIs for the data and is hosted on [render.com](http://render.com "render.com").
You can view it [here](https://countryinfo.onrender.com/ "here")
